# excel-utils

A curated collection of utilities for Microsoft Excel, organized by type.

## What's in this repository?

This repository contains reusable Excel utilities across three categories:

- **VBA** – Macro functions written in Visual Basic for Applications (VBA) to automate tasks and extend Excel's built-in capabilities.
- **JavaScript** – Office Scripts (TypeScript/JavaScript) that run in Excel for the web and Excel on Windows/Mac via the Automate tab.
- **LAMBDA** – Custom worksheet functions defined using Excel's [LAMBDA](https://support.microsoft.com/en-us/office/lambda-function-bd212d27-1cd1-4321-a34a-ccbf254b8b67) function, requiring no VBA or scripting knowledge.

## Folder structure

```
excel-utils/
├── vba/          # VBA macros and functions
├── javascript/   # Office Scripts (JavaScript/TypeScript)
└── lambda/       # Excel LAMBDA functions
```
